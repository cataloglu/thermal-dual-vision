# Contributing to Smart Motion Detector

## Geliştirme Süreci

1. **Issue Aç**: Önce bir issue açarak önerinizi tartışın
2. **Fork & Branch**: Repo'yu fork edin ve feature branch oluşturun
3. **Kod Yazın**: Değişikliklerinizi yapın
4. **Test Edin**: `pytest` ile testleri çalıştırın
5. **Dokümantasyon**: Gerekirse dokümanları güncelleyin
6. **PR Gönderin**: Pull request açın

## Kod Standartları

### Python
- PEP 8 uyumlu
- Type hints kullanın
- Docstring'ler ekleyin

### TypeScript/React
- ESLint kurallarına uyun
- Functional components kullanın
- Type safety öncelikli

## Commit Mesajları

```
feat: Yeni özellik ekle
fix: Hata düzelt
docs: Dokümantasyon güncelle
refactor: Kod iyileştirme
test: Test ekle/düzelt
chore: Bakım işleri
```

## Test Gereksinimleri

- Yeni özellikler için unit test ekleyin
- Integration test'ler önemli endpoint'ler için
- UI değişiklikleri için smoke test

## Dokümantasyon

Değişiklikleriniz şunları etkiliyorsa dokümantasyonu güncelleyin:
- API endpoint'leri → `docs/API_CONTRACT.md`
- UI bileşenleri → `docs/DESIGN_SYSTEM.md`
- Mimari değişiklikler → `docs/ARCHITECTURE.md`

## Sorular?

Issue açarak sorabilirsiniz!
