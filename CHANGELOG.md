# Changelog

## [2.0.0] - 2026-01-20

### Added
- Initial MVP documentation
- Complete API contract with all endpoints
- Design system with dark theme
- Architecture documentation
- Docker setup (compose + Dockerfiles)
- Frontend boilerplate (React + Vite + TypeScript + Tailwind)
- Backend skeleton (FastAPI + health endpoints)
- Basic tests
- Development setup guide
- Security guidelines
- Environment variables documentation

### Scope
- Person-only detection (YOLOv8n/s)
- Multi-camera support (color/thermal/dual)
- Event-based recording (collage/gif/mp4)
- Live view (MJPEG/WebRTC)
- Zone/ROI configuration
- Optional AI summary (OpenAI)
- Telegram notifications
- Retention policy + disk cleanup

### Out of Scope (MVP)
- Generic object detection
- Face recognition
- Continuous recording
- Multi-user authentication
